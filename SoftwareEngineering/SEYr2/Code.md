A code $C$ is any non-empty subset of $\mathbb F_2^n$

we say $C$ has length $n$

### [[Hamming Weight]] of a Code C
$$ h(C) = \min\;\; d(\vec x, \vec y) $$
where $\vec x, \vec y \in C$

### t Error Correcting
A coded C is t error correcting if and only if:
$$ h(C)\geq 2t+1 $$
