- [[UML]] defines an event as a specific occurence (in time and space) of a stimulus that can trigger transitions in state machines
- There are four types of event, each of which has different semantics
	- [[Call event]]
	- [[Signal Event]]
	- [[Change Event]]
	- [[Time Event]]
